# 355FinalProject

This project visualizes accessibility to essential services across British Columbia using Proximity Measures Database, 2021 from Statistique Canada

Note: This project is deployed using Firebase Hosting.

## Live Website (firebase hosting)

View the full interactive project here:
https://iat355-finalproject.web.app

## Dataset (firebase hosting)

The dataset is not included in this repository due to its large file size.
You can access it here:
https://iat355-finalproject.web.app/PMD-en.csv

## Running Locally

If you don't want to use the live firebase website link and want to run the website locally on your computer. Follow these steps:

1. Download the dataset from the dataset link above
2. Place the downloaded dataset into the public folder like this: public/PMD-en.csv
3. Start a local server (i.e. live server)
4. Open the project in your browser

## Deployment and updating website (For team members)

To deploy updates, enter this into the project's terminal:
firebase deploy
